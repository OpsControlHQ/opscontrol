# Roadmap

OpsControl is an early-stage infrastructure/security product. The roadmap is organized by product maturity rather than fixed public dates.

## Available

The following capabilities are available or represented in the current product foundation:

- Vault-backed access metadata.
- Infrastructure access metadata management.
- Service and port visibility.
- Controlled access workflow foundations.
- Access event tracking.
- AI-assisted query routing.
- Voice-to-text operational query support.
- Log and operational context assistance.

## In Progress

The following areas are currently being developed or hardened:

- unified context flow between InfraConnect and Workflow-AI;
- improved access workflow hardening;
- stronger separation between AI assistance and access control;
- documentation-grounded assistance;
- better operational context quality;
- clearer audit event modeling;
- improved product packaging for self-hosted environments.

## Planned

The following areas are planned as the product matures:

- temporary credential patterns;
- dynamic access patterns where supported by target systems;
- web dashboard;
- proactive alerting with engineer review;
- deeper knowledge base and RAG workflows;
- richer service visibility;
- broader policy and approval workflows.

## Product Direction

OpsControl will continue to develop around a strict separation of responsibilities:

- **InfraConnect** handles controlled access, credential workflows, visibility, and access accountability.
- **Workflow-AI** helps engineers understand operational context, logs, documents, and queries.

The long-term goal is not autonomous infrastructure control. The goal is better operational context and safer access workflows with the engineer still in control.

## Roadmap Boundaries

The roadmap does not imply:

- automatic privileged changes;
- AI-controlled infrastructure;
- AI access to secrets;
- replacement of engineer approval;
- unsupported maturity claims;
- public source code availability.

OpsControl remains a private, self-hosted infrastructure/security product.
